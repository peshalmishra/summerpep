# SummerPep

This repository contains a series of Python learning notebooks for the SummerPep course.

## Contents

- `day1/python_intro.ipynb` - Introduction to Python basics.
- `day2/conditional_statements.ipynb` - Conditional logic in Python.
- `day3/for_loop.ipynb` - Working with `for` loops.
- `day4/functions.ipynb` - Defining and using functions.
- `day4/nestedloop.ipynb` - Nested loop practice.
- `day4/whileloop.ipynb` - Using `while` loops.
- `day5/practice_nested.ipynb` - Nested practice exercises.

## Usage

Open the notebooks with Jupyter Notebook or JupyterLab.

```bash
jupyter notebook
```

Then navigate to the `dayX` folder you want to open.
